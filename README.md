## What I'm building

### ⚽ Gangs League
Competition management platform for amateur football-7 leagues in Melilla. Born from the frustration of watching real leagues run on WhatsApp groups and spreadsheets.

**Stack:** NestJS · Prisma · PostgreSQL · React · Docker · Cloudflare R2

Not a tutorial project. I've dealt with role-based guards, ownership patterns, TDD, injectable time dependencies, image storage, and the usual production headaches. Still going.
